# OpenClaw Dev Tooling v1.0

This repository provides automated workflows for managing Git worktrees and PR releases.

## Setup Instructions
1. Clone the repository to your local machine.
2. Ensure you have a `.env` file in the root directory containing your `DEVELOPER_API_KEY`.
3. Use your AI assistant (Copilot, Gemini, etc.) to analyze the project logic.
